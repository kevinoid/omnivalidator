Omnivalidator
=============

This is an add-on for Firefox (and compatible browsers) which eases the
process of validating resources (typically pages) against a set of
user-configurable validation tools.

This add-on currently supports the following features:
 * Displays validation errors in an in-browser panel based on the Error
   Console with clickable links to the location of the error.
 * Support for any number of local or remote validators
 * Fully localizable
